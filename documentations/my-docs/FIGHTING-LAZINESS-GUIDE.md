## ⚡ THE ANTI-LAZINESS SYSTEM

### Rule #1: **Break It Down to Ridiculously Small Tasks**

**DON'T SAY:** "I need to finish the frontend" ❌  
**SAY:** "I will implement the 'Start Simulation' button" ✅

**Why it works:** Your brain doesn't fear small tasks. It fears BIG SCARY TASKS.

### Rule #2: **The 2-Minute Rule**

If a task takes less than 2 minutes, **DO IT NOW.**

Examples:
- Create empty API function → 2 minutes
- Add a button to UI → 2 minutes  
- Write one test case → 2 minutes
- Fix one typo in code → 2 minutes

**Momentum builds from tiny wins.**

### Rule #3: **Pomodoro Technique (Modified for Programmers)**

```
1. Set timer for 25 minutes
2. Code/write with ZERO distractions
3. Take 5-minute break (stretch, water, NOT social media)
4. Repeat 4x
5. Take 15-30 minute break
```

**Download:** [Pomodoro Timer (any simple one)](https://pomofocus.io/)

### Rule #4: **Dopamine Reset**

Your brain is addicted to:
- Social media scrolling
- YouTube/TikTok
- Gaming
- Reddit/Twitter

**THE CURE:**
- **Delete** social media apps from phone (yes, really)
- Use browser extensions to block distracting sites during work hours
- Put phone in another room
- Set "Do Not Disturb" mode

**Recommended Tools:**
- **Cold Turkey** (Windows/Mac) - blocks websites
- **LeechBlock** (Firefox/Chrome) - blocks sites by schedule
- **Forest App** - gamify focus time

### Rule #5: **Accountability Partner**

Find someone who will check your progress:
- Classmate working on their thesis
- Friend who understands deadlines
- Online study buddy

**How to use:**
1. Share your daily task list with them
2. Report completed tasks every evening
3. They call you out if you slack off

**Pro tip:** Use Discord/WhatsApp for quick check-ins.

## 🧠 MENTAL TRICKS

### When You Feel Overwhelmed

**Use the "5-4-3-2-1" Grounding Technique:**
1. Name 5 things you can see
2. Name 4 things you can touch
3. Name 3 things you can hear
4. Name 2 things you can smell
5. Name 1 thing you can taste

**Then ask yourself:** "What's the SMALLEST thing I can do right now?"

### When You Don't Want to Start

**The "Just 2 Minutes" Hack:**
1. Tell yourself: "I'll just work for 2 minutes"
2. Set timer for 2 minutes
3. Start coding
4. When timer rings, you'll usually want to continue
5. If not, take a break and try again in 10 minutes

### When You Want to Give Up

**Remember:**
- This is temporary pain (1 month)
- You've survived harder things
- Your future self will thank you
- Failing is worse than struggling
- You're NOT alone - everyone struggles

**Emergency Motivation:**
1. Imagine presenting a complete, working project
2. Imagine your dosen saying "Bagus sekali!"
3. Imagine the relief after submitting
4. NOW GET BACK TO WORK
